# Srikanth Namoju - Personal Portfolio

A modern, responsive personal portfolio website showcasing my projects, skills, and professional journey as a BTech student specializing in Cloud Technology and Mobile Applications.

![Portfolio Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Scroll-triggered animations for enhanced user experience
- **Dynamic Greeting** - Time-based greeting (Good Morning/Afternoon/Evening)
- **Interactive Navigation** - Sticky navigation bar with smooth scrolling
- **Project Showcase** - Detailed presentation of academic and personal projects
- **Certificate Gallery** - Interactive certification viewer with overlay
- **Contact Section** - Easy-to-access contact information

## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SrikanthNamoju63/Srikanth-Portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Srikanth-Portfolio
   ```

3. **Open the project**
   - Simply open `index.html` in your web browser
   - Or use a local server like Live Server in VS Code

## Customization

### Changing Colors
Edit the CSS variables in `Polio.css`:
```css
/* Example: Change accent color */
background: linear-gradient(135deg, #your-color-1, #your-color-2);
```

### Adding Projects
Add a new project section in `index.html`:
```html
<div id="Portfolio" class="scroll-animation">
    <h3 id="para2">Your Project Title</h3>
    <p id="disc3">Your project description...</p>
</div>
```

### Updating Personal Information
Edit the About Me section in `index.html` with your details.
